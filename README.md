# Responsive Layout with CSS Grid

Página web estruturada utilizando CSS Grid para organização de layout, incluindo header, menu de navegação, conteúdo principal e rodapé.

## 🚀 Funcionalidades
- Layout estruturado com CSS Grid
- Menu de navegação horizontal com efeito hover
- Divisão de conteúdo em colunas (aside e conteúdo principal)
- Estilização visual com cores e espaçamentos

## 🛠 Tecnologias
- HTML
- CSS
